# Java Console Calculator

## About

This project is a simple console-based calculator developed in Java. It performs basic arithmetic operations by taking input from the user through the console.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero
* Menu-driven program
* Allows multiple calculations until the user chooses to exit

## Requirements

* Java JDK
* Any Java IDE or VS Code

## How to Run

Compile the program:

```bash
javac Calculator.java
```

Run the program:

```bash
java Calculator
```
